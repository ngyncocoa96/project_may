RMS UX Design Mockups

Travel Agent Record Management System: Python customtkinter GUI with green light theme

Overview

These mockups represent the visual layout of the final Travel Agent Record Management System GUI. The interface uses a clear sidebar and main record table so users can create, update, delete, search, sort, refresh, and export records from one central screen.

The system supports client, airline, and flight records. The sidebar gives users direct access to the main actions: Add Client, Add Airline, Add Flight, Delete Record, Update Record, and Export to CSV. The main area contains the search bar, Refresh All button, sortable records table, and vertical scrollbar for handling larger record lists.

UX Evaluation Summary

  • Clarity: sidebar labels clearly describe every action, including the Export to CSV button.
  • Consistency: create and update windows follow the same label-entry-save structure across all record types.
  • Error prevention: validation checks cover empty fields, date format, letter-only fields, number-only fields, phone format, and database existence for linked IDs.
  • Safety: delete confirmation and read-only ID field in the update window prevent accidental data loss or identity corruption.
  • Feedback: messageboxes support user understanding through success, validation, format, security, search, export, and system responses.
  • Efficiency: double-click to update, Enter key in search, Refresh All, column sorting, and partial search all reduce the number of steps for common tasks.
  • Structure: the interface keeps user actions organised while maintaining a consistent flow across the main window and popup forms.

Future UX Improvements

  • Show the client name next to Client ID in the Flight form as helper text.
  • Add a status bar at the bottom of the main window showing the total number of loaded records.
  • Consider a filter dropdown to show only clients, airlines, or flights at once.
  • Add a short save-status message or visual indicator after records are created, updated, deleted, or exported.
